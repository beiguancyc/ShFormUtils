# ShFormUtils
android表单映射框架
#Download

Download the latest JAR or grab via Maven:
~~~xml
<dependency>
  <groupId>com.squareup.okhttp3</groupId>
  <artifactId>okhttp</artifactId>
  <version>3.5.0</version>
</dependency>
~~~
or Gradle:
~~~gradle
compile 'com.squareup.okhttp3:okhttp:3.5.0'
~~~
